# Support

Banban is an early-stage free and open-source project.

## Before Asking For Help

Please check:

- `README.md`
- `docs/device-test-plan.md`
- `docs/release-checklist.md`
- Existing GitHub issues, after the repository is created

## Good Support Requests Include

- What you were trying to do.
- What happened.
- Platform: iOS, Android, or Web.
- Whether you used Expo Go or a development build.
- Whether the issue happens with no DeepSeek Key configured.

Do not include API keys, passwords, tokens, or private user data.

## Scope

The project can help with:

- App setup
- Local development
- DeepSeek API wiring
- Local data behavior
- Notifications
- Store preparation

The project cannot provide:

- Personal DeepSeek billing support
- Medical, legal, financial, or psychological advice
- Help generating illegal or harmful content
