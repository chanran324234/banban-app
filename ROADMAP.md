# Roadmap

## 0.1.0 MVP

- Expo + React Native app shell
- DeepSeek user-owned API Key
- AI chat with structured task and memory output
- Daily plan, night review, and project breakdown workflows
- Local tasks, memories, reminders, setup state, and chat persistence
- Local data export and clear data controls
- App icon, splash screen, screenshots, and store copy draft
- EAS build configuration

## 0.2.0 Mobile Validation

- iOS real-device notification testing
- Android real-device notification testing
- DeepSeek Key secure storage validation
- Store screenshot refresh from real devices or simulators
- Privacy policy and terms hosted at stable URLs
- First TestFlight / internal Android build

## 0.3.0 Usability

- Editable generated tasks before saving
- Memory editing and categories
- Workflow history
- Better empty states
- Local search across tasks, memories, and chat

## 0.4.0 Open Source Hardening

- Automated checks in GitHub Actions
- Contributor documentation refinements
- Security advisory setup
- Release notes process
- Issue triage labels and project board

## Later

- Optional self-hosted backend
- Optional encrypted sync
- Custom workflow templates
- Internationalization
