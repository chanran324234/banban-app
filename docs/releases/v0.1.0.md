# Banban v0.1.0

Release date: 2026-06-06

This is the first open-source MVP release of Banban.

## Highlights

- Free and open-source AI companion + workflow automation app.
- User-owned DeepSeek API Key; no shared API key is bundled.
- AI chat with structured task and memory extraction.
- Daily plan, night review, and project breakdown workflows.
- Local-first task, memory, reminder, setup state, and chat persistence.
- Local data export and clear data controls.
- Local reminders for tasks, daily planning, and night review.
- App icon, splash screen, notification icon, screenshots, and store listing draft.
- EAS build profiles for development, preview, and production.
- GitHub issue templates, PR template, security policy, support doc, roadmap, changelog, and CI check workflow.

## Validation

Passed:

- `npm run check`
- Expo dependency compatibility check
- TypeScript typecheck

## Known Limitations

- Web preview does not support system notifications.
- No account system or cloud sync exists yet.
- Store URLs, support email, and GitHub repository URLs still need final production values.
- iOS and Android notification behavior still needs real-device validation.

## Upgrade Notes

This is the initial release. No migration is required.
