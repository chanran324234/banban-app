# Banban v0.1.1

Release date: 2026-06-06

This patch release improves store-submission readiness and makes the repository easier to build from a fresh checkout.

## Highlights

- Pins EAS CLI as a development dependency.
- Adds npm scripts for EAS login, preview builds, production builds, and store submission.
- Adds a store-submission guide for App Store / Google Play preparation.
- Updates public store and privacy docs to remove the fake placeholder email.
- Bumps the app version to 0.1.1.
