# 伴办隐私政策

更新日期：2026-06-06

伴办是免费开源的 AI 陪伴 + 工作流自动化 App。第一版默认把任务、记忆和聊天记录保存在用户本机。

## 数据存储

- 任务、记忆、聊天记录：保存在本机。
- DeepSeek API Key：保存在系统安全存储中。
- Web 预览环境：使用浏览器 localStorage 作为开发兜底。

## 模型请求

用户配置自己的 DeepSeek API Key 后，聊天内容会发送到 DeepSeek API，用于生成回复、任务和记忆建议。

伴办不内置统一 API Key，也不替用户托管模型密钥。

## 用户控制

用户可以：

- 替换或删除 DeepSeek API Key。
- 导出本地数据。
- 清空本地任务、记忆和聊天记录。

## 联系方式

正式上架前需要补充开发者联系邮箱和开源仓库地址。
