# 伴办真机测试计划

更新日期：2026-06-06

## 目标

确认伴办在 iOS / Android 真机上可以完成核心 MVP 流程：

- 保存 DeepSeek API Key
- AI 聊天
- 工作流生成
- 本地任务、记忆、聊天持久化
- 数据导出 / 清空
- 本地通知提醒

## 准备

1. 安装依赖：

   ```bash
   npm install
   ```

2. 生成资产：

   ```bash
   npm run assets
   ```

3. 启动开发服务：

   ```bash
   npm start
   ```

4. 用 Expo Go 或开发构建在真机打开应用。

## iOS 测试项

- App 图标显示正确。
- 启动页显示正确。
- 设置页可以保存 DeepSeek API Key。
- 首页首次使用 checklist 会显示 Key 配置状态。
- 关闭并重新打开 App 后，Key 仍可读取。
- 聊天页能使用 DeepSeek 回复。
- 工作流页可以生成每日计划、晚间复盘、项目拆解。
- 成功运行工作流后，首页 checklist 会更新。
- 任务页可以切换任务状态。
- 任务、记忆、聊天记录重启后仍存在。
- 设置页可以导出 JSON。
- 设置页可以清空本地数据和 Key。
- 每天 9 点计划提醒可以申请通知权限。
- 每天 21 点复盘提醒可以申请通知权限。
- 单个任务 1 小时后提醒可以创建。

## Android 测试项

- Android 13+ 会请求通知权限。
- 通知栏图标使用白色透明图标，不显示彩色方块。
- App 图标和 adaptive icon 显示正确。
- 其余功能同 iOS 测试项。

## DeepSeek 测试项

使用一个低风险测试 Key。

测试提示：

- “我想把伴办做成可以上架的免费开源应用，帮我拆下一步。”
- “帮我生成今天的计划。”
- “帮我把这个项目拆成里程碑。”

预期：

- 聊天回复可读。
- 结构化任务会自动进入任务页。
- 重要信息可以保存为记忆。
- 工作流结果可以写入聊天。

## 已知限制

- Web 预览不支持系统通知。
- 当前没有账号系统，所有数据默认本地保存。
- 当前没有云同步。
- 当前未接真实应用商城固定隐私政策 URL。
