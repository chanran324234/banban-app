## 长描述

伴办是一个免费开源的 AI 陪伴 + 工作流自动化应用。

它不是单纯聊天机器人，也不是冷冰冰的待办工具。你可以把正在想、正在卡、正在推进的事告诉伴办，它会帮你整理成今日计划、任务拆解、项目里程碑和晚间复盘。

核心能力：

- AI 聊天：把模糊想法整理成可执行下一步。
- 每日计划：生成今日重点、时间块和风险提醒。
- 晚间复盘：整理完成项、卡点和明天第一步。
- 项目拆解：把项目目标拆成里程碑、风险和下一批任务。
- 长期记忆：记录目标、偏好和项目方向，并支持查看和删除。
- 本地数据控制：支持导出和清空任务、记忆、聊天记录。
- 本地提醒：支持任务提醒、早间计划提醒和晚间复盘提醒。

伴办免费开源，不内置统一模型密钥。用户可以在设置页填入自己的 DeepSeek API Key。任务、记忆和聊天记录默认保存在本机。
